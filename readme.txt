AI Content Toolkit是一个功能强大的智能内容管理插件，集成AI文章优化、SEO分析、网站卡片等10大功能，让WordPress内容管理更智能。

== Description ==

AI Content Toolkit将10个实用工具完美整合在一起，您不再需要安装多个插件来满足不同的网站需求，只需要这一个插件就能解决所有问题！

### 主要功能

**🤖 AI文章优化工具**
- **AI自动分类**：智能分析文章内容，自动选择最合适的分类
  - 三种智能模式：草稿总是分类、智能首次分类、草稿首次分类
  - 基于WordPress默认分类检测，避免覆盖用户手动选择
  - 支持自定义提示词，提升分类准确性
- AI摘要生成：智能提取文章精华，自动生成高质量摘要
  - 统一三种模式：与AI分类相同的智能生成策略
  - 支持草稿状态持续优化，发布后保持稳定
- AI标签优化：精准内容标记系统，长尾关键词挖掘
  - 智能标签生成：根据内容自动推荐相关标签
  - 与分类、摘要统一的三种模式管理
- AI SEO分析报告：15维度内容分析，竞争对手对比，智能优化建议

**🌐 网站功能增强**
- 网站卡片展示：自动将链接转换为美观卡片
- 友情链接管理：完整的链接管理系统
- Cookie同意通知：GDPR合规的隐私保护
- REST代理修复：智能修复WordPress连接问题

**📊 实用工具**
- 年龄计算器：精确的时间计算工具
- 物品管理系统：个人资产管理工具
- AI设置管理：多服务商智能管理

== Installation ==

1. 通过WordPress后台安装：
   - 登录WordPress后台
   - 进入"插件" → "安装插件"
   - 搜索"AI Content Toolkit"
   - 点击"现在安装"并激活

2. 手动安装：
   - 下载插件压缩包
   - 进入"插件" → "上传插件"
   - 选择文件并上传
   - 激活插件

== Frequently Asked Questions ==

= 是否需要付费？ =

AI Content Toolkit完全免费开源，无任何隐藏费用。

= 需要配置AI服务吗？ =

插件支持多个AI服务商，包括DeepSeek和硅基流动。用户可以根据需要配置API密钥，也可以使用基础功能。

= 插件兼容哪些WordPress版本？ =

兼容WordPress 5.0及以上版本，推荐使用PHP 7.4+。

= 是否支持多语言？ =

是的，插件支持多语言，完美处理中英文混合内容。
