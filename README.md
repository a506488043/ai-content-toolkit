## 📦 安装

### 自动安装（推荐）
1. 登录WordPress后台
2. 进入"插件" → "安装插件"
3. 搜索"AI Content Toolkit"
4. 点击"现在安装"并激活

### 手动安装
1. 下载插件压缩包
2. 进入"插件" → "上传插件"
3. 选择文件并上传
4. 激活插件

## 📚 使用文档

### AI自动分类使用指南

**三种智能模式**：
1. **草稿总是分类**（推荐）：草稿状态每次保存都AI分类，适合持续优化的内容
2. **智能首次分类**：只有默认文章分类时才AI分类，避免覆盖用户手动选择
3. **草稿首次分类**：只对草稿第一次保存时AI分类，适合一次性生成

**功能特点**：
- 🤖 智能分析文章标题和内容，自动推荐最合适的分类
- 🎯 基于WordPress默认分类检测，尊重用户手动选择
- ⚡ 草稿状态下实时优化，发布后保持分类稳定
- 📝 支持自定义AI提示词，提升分类准确性
- 🔧 可设置置信度阈值和最大分类数量

**设置路径**：
- WordPress后台 → AI内容工具包 → 工具箱设置 → AI自动分类

### 统一的AI内容管理模式

AI分类、摘要、标签功能现在采用统一的**三种智能模式**：
- **模式一致性**：所有AI功能使用相同的触发逻辑
- **用户友好**：避免AI覆盖用户手动创建的内容
- **灵活配置**：根据使用习惯选择最适合的模式
